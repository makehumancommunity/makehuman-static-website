These pages contain code documentation and general hints for people who want to help develop MPFB.

## Getting the code

The code repository is at https://github.com/makehumancommunity/mpfb2. The addon zip on the extension platform
is in practice a simple archive of the "mpfb" [subdirectory of the src folder](https://github.com/makehumancommunity/mpfb2/tree/master/src/mpfb). 

The code is editor agnostic, although PyDev has been used when writing it. 

## Introductory material

Most of the technical reference documentation [is available in the git repo](https://github.com/makehumancommunity/mpfb2/blob/master/docs/index.md).
A good place to start is by reading the description of the [code structure](https://github.com/makehumancommunity/mpfb2/blob/master/docs/code-structure.md).
